description: リリース前チェックリスト
---

リリース前の確認を順番に実行してください。

## チェックリスト

1. **テスト実行**: `npm test` を実行し、全テストが通ることを確認
2. **ビルド確認**: `npm run build` を実行し、エラーがないことを確認
3. **変更差分の確認**: `git log --oneline main..HEAD` で含まれるコミットを一覧表示
4. **docs 同期確認**: 変更内容に応じて以下のドキュメントが更新されているか確認
   - `CLAUDE.md` の対応表や規約
   - `.claude/rules/` のパス固有ルール
   - `docs/specs/` の仕様書
5. **Codex レビュー（任意）**: 変更が中規模以上の場合、`/codex-review` でレビューを実施

## 全チェック通過後

「リリース準備完了」と報告する。ユーザーが承認したら `npm run release` を実行。
