## フレームシステム

すべての電文は「フレーム」で囲まれて表示されます。フレームのスタイルは情報の重大度（`FrameLevel`）に応じて切り替わります。

### フレームレベル一覧

| レベル | 罫線文字 | 色 | 用途 |
|--------|----------|----|------|
| `critical` | `╔ ╗ ╚ ╝ ═ ║ ╠ ╣` (二重線) | 赤 | 震度6弱以上、EEW警報、大津波警報、南海トラフ巨大地震警戒 |
| `warning` | `╔ ╗ ╚ ╝ ═ ║ ╠ ╣` (二重線) | 黄 | 震度4以上、EEW予報、津波警報、南海トラフ注意/調査中 |
| `normal` | `┌ ┐ └ ┘ ─ │ ├ ┤` (単線) | シアン | 震度3以下の地震情報、長周期階級2 |
| `info` | `┌ ┐ └ ┘ ─ │ ├ ┤` (単線) | グレー | テキスト情報、調査終了、長周期階級1以下 |
| `cancel` | `┌ ┐ └ ┘ ─ │ ├ ┤` (単線) | 緑 | 取消報 |

### フレーム構造

```
╔══════════════════════════════════════════════════════════╗   ← frameTop
║ (コンテンツ行)                                          ║   ← frameLine
╠══════════════════════════════════════════════════════════╣   ← frameDivider
║ (コンテンツ行)                                          ║   ← frameLine
╚══════════════════════════════════════════════════════════╝   ← frameBottom
```

デフォルト幅: 60 文字（`config.tableWidth` で変更可能）

## 地震情報 (VXSE51/52/53/61)

**関数**: `displayEarthquakeInfo(info: ParsedEarthquakeInfo)`
**パーサ**: `parseEarthquakeTelegram()`

### 電文タイプ

| コード | 名称 |
|--------|------|
| VXSE51 | 震度速報 |
| VXSE52 | 震源に関する情報 |
| VXSE53 | 震源・震度に関する情報 |
| VXSE61 | 顕著な地震の震源要素更新のお知らせ |

### フレームレベル判定

| 条件 | レベル |
|------|--------|
| `infoType` = 取消 | `cancel` (緑) |
| 最大震度 ≥ 6弱 | `critical` (赤) |
| 最大震度 ≥ 4 | `warning` (黄) |
| その他 | `normal` (シアン) |

### 表示構成 (上から順)

| セクション | 条件 | 内容 |
|-----------|------|------|
| テスト電文バッジ | `isTest = true` | ` テスト電文 ` (マゼンタ背景) |
| タイトル行 | 常に表示 | `{電文タイプ名} [{コード}]  {情報の種類}` |
| ── 区切り ── | | |
| カード行 | 該当データがある場合 | `最大震度 {N}  │  長周期階級 {N}  │  M{x.x}  │  深さ {N}km  │  {津波状況}` |
| ── 区切り ── | | |
| 震源詳細 | `earthquake` がある場合 | 震源地名 / 発生時刻 / 座標 |
| 発表時刻 | 常に表示 | `発表: {時刻}  {発表官署}` (グレー) |
| ── 区切り ── | | |
| ヘッドライン | `headline` がある場合 | 太字白 |
| ── 区切り ── | | |
| 震度一覧 | 震度データがある場合 | 震度7→1の降順、地域名をカンマ区切り。長周期階級1以上の地域には `[長周期N]` 付記 |
| ── 区切り ── | | |
| 津波情報 | `tsunami` がある場合 | 津波予報テキスト |

### 津波状況の短縮表示

| 元テキスト | 表示 | 色 |
|-----------|------|-----|
| 「心配はありません」含む | `津波なし` | 緑 |
| 「注意」含む | `津波注意` | 黄 |
| 「警報」含む | `津波警報` | 赤 |
| その他 | テキスト先頭10文字 | 白 |

### 表示構成 (上から順)

| セクション | 条件 | 内容 |
|-----------|------|------|
| バナー | 常に表示 | 3行の背景色バナー。取消=緑, 警報=赤, 予報=黄 |
| テスト電文バッジ | `isTest = true` | ` テスト電文 ` (マゼンタ背景) |
| PLUM法ラベル | `maxIntChangeReason = 9` | `PLUM法 による予測震度変化` (マゼンタ) |
| ── 区切り ── | | |
| カード行 | 取消以外 | `{infoType}  │  最大予測震度 {N}  │  長周期階級 {N}  │  M{x.x}  │  深さ {N}km` |
| 同時発生注記 | `activeCount ≥ 2` | `同時{N}件発生中` (黄) |
| 予測震度差分 | `diff.previousMaxInt` あり | `最大予測震度 {旧} → {新}` |
| ── 区切り ── | | |
| 仮定震源ラベル | `isAssumedHypocenter` | `仮定震源要素 (震源未確定・PLUM法による推定)` (マゼンタ) |
| 震源詳細 | `earthquake` がある場合 | 震源地名（変更時は `(変更)` 付記, シアン） / 発生時刻 / 規模（差分あり: `M旧 → M新`） / 深さ（差分あり: `旧 → 新`） |
| 発表時刻 | 常に表示 | グレー |
| ── 区切り ── | | |
| ヘッドライン | `headline` がある場合 | 太字白 |
| 取消メッセージ | 取消時 | `この地震についての緊急地震速報は取り消されました。` (緑) |
| ── 区切り ── | | |
| 予測震度一覧 | 非取消時 | `震度{N}: {地域名}` + オプションタグ |
| 到達地域一覧 | 到達地域あり | `既に主要動到達と推測:` (赤) + 地域名リスト |
| ── 区切り ── | | |
| 最終報テキスト | `nextAdvisory` あり | シアン |
| EventID | `eventId` あり | グレー |

### 地域名に付くタグ

| タグ | 色 | 条件 |
|------|----|------|
| `[PLUM]` | マゼンタ | `area.isPlum = true` |
| `[到達]` | 赤 | `area.hasArrived = true` |
| `[長周期N]` | 階級色 | `area.lgIntensity` が 1 以上 |

### 仮定震源要素時の制限

`isAssumedHypocenter = true` の場合、以下は非表示:
- カード行のマグニチュード・深さ
- 震源詳細の規模・深さ
