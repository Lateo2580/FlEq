インストール

GETTING STARTED

1

インストール

npm install -g @sayue_ltr/fleq

または単発で実行:

npx @sayue_ltr/fleq --help
2

初期設定

対話形式で API キーや受信設定をまとめて行えます。

fleq init
3

起動

fleq

前提条件

  • Node.js 18 以上が必要です
  • dmdata.jp のアカウントと API キーが必要です
  • API キーには socket.start 権限と受信区分の telegram.get.* 権限を付与してください
  • 情報の受信には dmdata.jp の有料契約が必要です

対応 OS
